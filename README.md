# Stellar Burger (SPA)

[Макет](<https://www.figma.com/file/vIywAvqfkOIRWGOkfOnReY/React-Fullstack_-Проектные-задачи-(3-месяца)_external_link?type=design&node-id=0-1&mode=design>)

## Этапы работы:
1. Разработала одностраничное приложение (SPA) на React с клиентским роутингом, глобальным состоянием и авторизацией пользователей. Использовала готовые UI-компоненты, размещённые в директории `src/components`.

2. Реализовала взаимодействие с сервером: настроила получение и обработку данных через `Redux` и глобальный `store`. API-эндпоинты были заранее описаны в `utils/burger-api.ts`.

3. Настроила protected routes и реализовала личный кабинет пользователя с контролем доступа.

4. Покрыла приложение unit-, integration- и e2e-тестами.

5. Использовала Storybook для разработки, изоляции и проверки UI-компонентов.


## Важно:

Для корректной работы запросов к серверу необходимо добавить переменную BURGER_API_URL в окружение. Сама ссылка находится в файле `.env.example`.

Стек: `React`, `TypeScript`, `React Router`, `Jest`, `Cypress`, `Storybook`
